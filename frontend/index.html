DDUK ERP

팀 프로젝트 시작 화면

로그인 후 역할별 메인 페이지로 이동해 각 도메인 작업을 바로 시작할 수 있다.

Admin

인증, 권한, 공통 운영 기능 작업 시작

HR

직원, 근태, 급여, 비용 관리 화면 작업 시작

Inventory

거래처, 재고, 발주, 입출고 흐름 작업 시작

작업 시작 전 체크

  • 백엔드가 `http://localhost:8080`에서 실행 중인지 확인
  • 자기 도메인 `admin`, `hr`, `inventory` 경계를 먼저 확인
  • 공용 계약 변경 시 문서를 먼저 갱신
